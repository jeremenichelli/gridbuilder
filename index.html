Grid Builder by Jeremias Menichelli

Choose a number of columns for your grid and the space between the columns in percent to create a fluid grid. Each columns have the .column class and another one for the width it needs to cover, for example .column-2. All columns must be wrapped in a .row div element. I'm also using Eric Meyer's CSS Reset to make sure the grid looks the same in all browsers.
